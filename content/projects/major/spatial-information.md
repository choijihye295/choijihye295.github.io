## Java 프로젝트

여기에는 Java 프로젝트에 대한 세부 정보가 들어갑니다. 이 프로젝트는 Java를 사용하여 애플리케이션을 개발하는 과정에서의 경험과 결과물을 포함합니다.

### 프로젝트 목표
- Java 기반의 애플리케이션 개발
- 사용자 인터페이스 설계
- 데이터베이스 연동

### 기술 스택
- Java
- Spring Boot
- MySQL
